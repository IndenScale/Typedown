# VS Code Extension

Typedown officially provides a VS Code extension to deliver the ultimate writing experience.

## Key Features

*   **Syntax Highlighting**: Specific highlighting for `entity` code blocks.
*   **Real-time Diagnostics**: Display Pydantic validation errors directly in the editor.
*   **Smart Completion**: Auto-suggest field names based on imported class definitions.
*   **Documentation Navigation**: Ctrl+Click to jump to class definition or `former` object definition.
