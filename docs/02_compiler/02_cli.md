# CLI Command Line Tool

`td` is the command-line entry point for Typedown.

## Common Commands

*   `td init <project_name>`: Initialize a new Typedown project structure.
*   `td check`: Run the complete analysis and validation process, reporting errors.
*   `td build --output <dir>`: Compile documentation and output a static site (TBD).
*   `td materialize [file_path]`: Expand "syntactic sugar" in specified files (or all files) into full code blocks.

## Argument Description

Use `td --help` to see all available options.
